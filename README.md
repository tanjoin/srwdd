# SRWDD パイロットデータ比較・ランキングツール

このWebアプリは「スーパーロボット大戦DD」のパイロットデータ（CSV形式）をアップロードし、
・パイロット同士のステータス比較
・各種ステータスランキング表示
を行うツールです。

## 主な機能

- **CSVアップロード**: 独自フォーマットのCSVファイルをアップロードしてデータを読み込みます。

## 使い方

### 1. セットアップ

1. [Node.js](https://nodejs.org/) と `npm` をインストールしてください。
2. プロジェクトのルートディレクトリで以下を実行：

    ```bash
    npm install
    ```

### 2. 開発サーバー起動

```bash
npm run dev
```

ターミナルに表示された `http://localhost:xxxx` をブラウザで開きます。

## 画面・操作説明

1. 「ファイルを選択」ボタンでCSVファイルをアップロード
2. パイロット一覧や各種ステータスが表形式で表示されます
3. スキル装備やデータ編集も可能です

## ビルド・デプロイ

本番用ビルド：

```bash
npm run build
```

ビルド成果物は `docs` ディレクトリに出力されます。

ローカルプレビュー：

```bash
npm run preview
```
