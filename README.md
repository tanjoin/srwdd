# SRWDD パイロットデータ比較・ランキングツール

このWebアプリは「スーパーロボット大戦DD」のパイロットデータ（CSV形式）をアップロードし、
・パイロット同士のステータス比較
・各種ステータスランキング表示
を行うツールです。

## 主な機能

- **CSVアップロード**: 独自フォーマットのCSVファイルをアップロードしてデータを読み込みます。
- **パイロット比較**: 2人のパイロットを選択し、各ステータスの合計値や内訳を比較できます。
- **ランキング表示**: 攻撃力、防御力、照準値、運動性など各種ステータスのランキングを表示します。

## 使い方

### 1. セットアップ

1. [Node.js](https://nodejs.org/) と `npm` をインストールしてください。
2. プロジェクトのルートディレクトリで以下を実行：

    ```bash
    npm install
    ```

### 2. 開発サーバー起動

```bash
npm run dev
```

ターミナルに表示された `http://localhost:xxxx` をブラウザで開きます。

### 3. CSVファイルの準備

`public/pilots.csv` にサンプルCSVがあります。独自のデータを使う場合は同じ形式で用意してください。

#### 必須ヘッダー例


```
id,name,series,攻撃力_基礎ステータス,攻撃力_基本スキル,攻撃力_特殊スキル,防御力_基礎ステータス,防御力_基本スキル,防御力_特殊スキル,照準値_基礎ステータス,照準値_基本スキル,照準値_特殊スキル,運動性_基礎ステータス,運動性_基本スキル,運動性_特殊スキル
```

#### データ例

```csv
id,name,series,攻撃力_基礎ステータス,攻撃力_基本スキル,攻撃力_特殊スキル,防御力_基礎ステータス,防御力_基本スキル,防御力_特殊スキル,照準値_基礎ステータス,照準値_基本スキル,照準値_特殊スキル,運動性_基礎ステータス,運動性_基本スキル,運動性_特殊スキル
1,ユニス・エアリー,オリジナル,432,1728,3726,396,1584,3218,73,110,257,73,110,225
2,相良宗介,フルメタル・パニック,432,1728,10530,372,1488,8835,67,101,798,84,126,1024
```

## 画面・操作説明

### 比較ページ（index.html）
1. 「ファイルを選択」ボタンでCSVファイルをアップロード
2. 2つのドロップダウンからパイロットを選択
3. 「比較」ボタンで、各ステータスの合計値や内訳を比較表示

### ランキングページ（ranking.html）
1. 「ファイルを選択」ボタンでCSVファイルをアップロード
2. 各ステータスのランキングが自動表示
